# **开始使用它之前**
1. 使用' DATABSE_SQL '文件夹中提供的SQL文件创建相应的数据库。  
2. 在' druid '下修改你的数据库配置(连接到数据库的帐户和密码)。在“/src/main/resources/”文件夹中。  
3. 由于项目直接提供源代码，并使用“Maven”来配置依赖项，请确保您在您的计算机上安装和配置Maven。如果你没有听说过Maven，相关知识请访问[Maven](https://maven.apache.org/)
